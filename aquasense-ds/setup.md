# Setup Instructions

## Download Dataset

O dataset utilizado neste projeto é muito grande para ser armazenado no repositório. Você precisa baixá-lo manualmente seguindo os passos abaixo:

### Passos:

1. Acesse o link do Google Drive:
   https://drive.google.com/file/d/1F_ZAV5mJLuz-C9FrG1h4VOw-1mYjAU92/view?usp=sharing

2. Clique em "Download" (ícone de seta para baixo no canto superior direito)

3. Extraia o arquivo baixado (se necessário)

4. Mova o arquivo `Combined_dataset.csv` para a pasta `dataset/`:
   ```bash
   mv /caminho/para/Combined_dataset.csv ./dataset/
   ```

Após seguir esses passos, o dataset estará disponível em `dataset/Combined_dataset.csv` e você poderá executar os notebooks normalmente.

## Nota

O arquivo dataset é ignorado pelo git (configurado em `.gitignore`)
